# Huerto Hogar – Carrito integrado (Atomic + Vite)
- Carrito basado en localStorage como el del profe (`products`).
- Botón “Añadir al Carrito” en cada ProductCard.
- Página `/carrito` con tabla, total, eliminar y limpiar.
